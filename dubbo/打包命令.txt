1. 源码下载
先从SVN：https://github.com/smatem/dubbo/trunk/dubbo（dubbo-admin已经打好dubbo-admin-2.5.4-SNAPSHOT.war WAR包）下载建立MAVEN项目
再从 Alibaba Dubbo gitbub URL：https://github.com/alibaba/dubbo 下载缺少的项目源码

2. 准备软件
安装配置Java，MAVEN环境变量，Eclipse集成开发工具，MAVEN项目扩展，Tomcat Web容器

3. CMD进入Dubbo根目录

4. 执行命令
MAVEN_OPTS= $MAVEN_OPTS -Xms256m -Xmx1024m -XX:MaxPermSize=128m -XX:ReservedCodeCacheSize=64m 具体大小，可根据自己的情况调整(测试环境不重要)
再次执行：mvn clean install -Dmaven.test.skip（Tomcat Ver 8.0.*, Java Ver 1.7）

5. 进入dubbo-admin/target/路径找到dubbo-admin-2.5.4-SNAPSHOT.war，将War包放入tomcat建立WEB控制台
   安装Zookeeper，找到WEB-INF/dubbo.properties，配置注册中心（zookeeper地址）
